DPS home page

The template for DPS home page is a mix of HTML and json data.

To add your service as a tile, and to the menus, look for the file data/dps-services.json in the data folder.

Add your services details in the correct alphabetical place in this file with a link and it will create the relevant tile and menu items.

If you have different versions of your prototype, use a dot slash relative url instead of hardcoding it eg. ./name-of-folder.

You can also change the prison, the user name or todays prison details in data/session-data-defaults.js if neccessary.
